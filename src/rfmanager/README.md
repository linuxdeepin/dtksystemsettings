## dtkrfmanager

rfkill interface binding for Qt5

## Dependencies

### build on debian/uos
requirements:

- qtbase5-dev
- cmake
- doxygen

### Build from source code

```bash
sudo apt install qt5-default doxygen cmake  
cmake -B build
cmake --build build/
cd build
sudo make install
```
