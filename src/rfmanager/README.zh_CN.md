## dtkrfmanager

是对 rfkill 接口进行封装，方便管理无线设备。

## 编译依赖

### 在debian/uos上构建

软件包依赖:

- qtbase5-dev
- cmake
- doxygen

### 从源代码构建

```bash
sudo apt install qt5-default qttools5-dev doxygen cmake  
cmake -B build
cmake --build build/
cd build
sudo make install
```
