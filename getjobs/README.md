## getjobs

Get Print Jobs module, used for get user print task information.

[中文文档](./README.zh_CN.md)

### Build dependencies

Project compile-time dependencies: cmake,gcc,g++,libcups-dev

```bash
sudo apt install cmake libcups-dev
```

## Installation

### Build from source code

```bash
sudo apt install cmake libcups-dev
cmake -B build
cmake --build build/
cd build
sudo make install
```
